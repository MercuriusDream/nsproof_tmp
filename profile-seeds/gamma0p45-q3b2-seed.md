# Approximate Compactified Profile Seed: gamma = 0.45, B = 1.0

This is an approximate low-order seed for the tail-factored axisymmetric profile ansatz

```text
psi = r^2 z q^(1/gamma) F(q,b),
Gamma = r^2 q^(1/gamma) G(q,b),
q = (1+r^2+z^2)^(-1/2),
b = z/sqrt(r^2+z^2).
```

It is not a validated profile and must not be used as proof of blow-up. It is a continuation seed for the next nonlinear solve.
